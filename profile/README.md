# Hy-Fi Studio

Hy-Fi Studio is a premier development platform dedicated to crafting exceptional plugins and mods for Hytale servers. Our team of experienced developers delivers enterprise-grade tools engineered for multiplayer environments, setting the industry standard for performance, reliability, and extensibility.

## Overview

All plugins are meticulously crafted for multiplayer servers and professional minigame networks. Each product represents the culmination of extensive development experience, delivering polished gameplay mechanics that server owners can confidently deploy to create premium player experiences.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **Hy-Fi HeadSteal** | A PvP mechanic where players collect heads from eliminated opponents as trophies or currency. |
| **Hy-Fi LifeSteal** | Health-based survival mode where killing other players transfers their life points to you. |
| **Hy-Fi Swords** | A dueling framework with ranked matchmaking and customizable sword combat mechanics. |
| **Hy-Fi Death Run** | Obstacle course minigame where runners avoid traps set by death players. |
| **Hy-Fi Spleef** | Classic arena game where players break blocks beneath opponents to eliminate them. |
| **Hy-Fi Hide and Seek** | Prop hunt-style minigame where hiders disguise as objects while seekers hunt them down. |
| **Hy-Fi Bridge** | Fast-paced bridging duels where players race to cross a void and score on the enemy side. |
| **Hy-Fi SkyBlock** | Island survival mode with resource management, upgrades, and multiplayer island features. |
| **Hy-Fi Pillars of Fortune** | Luck-based minigame where players ascend pillars with randomized rewards and hazards. |
| **Hy-Fi SkyWars** | Aerial PvP battle royale on floating islands with loot chests and shrinking playable area. |
| **Hy-Fi BedWars** | Team-based strategy game where players protect their bed while attempting to destroy others. |
